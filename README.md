# Run application 

- Pre-Requisite : Docker should be installed
- Follow steps below : 

        $ docker-compose up -d 
- Go to localhost:3000 to see the landing page
- The Node API Server will be running at localhost:5000/api/users
- After Testing : 

        $ docker-compose down
